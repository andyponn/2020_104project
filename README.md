# 找個實習吧PYTHON爬蟲專案
## 更新日期：114.04.12
###### 本專案旨在透過爬蟲技術爬取104, yes123, 1111網站各實習職缺，透過整理職缺成自定義標籤彙整成一查詢系統，方便使用者透過此系統能一鍵瀏覽各大網站職缺，幫助大學生一手掌握實習資訊。
